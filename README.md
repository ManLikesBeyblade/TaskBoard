# TaskBoard

### Ideas

- Basically trello
- Users can create boards with lists that contain tasks; tasks can be moved between lists

### Classes
- Task
  - String name, description
- List
  - String name, description
  - ArrayList<Task> tasks
- Board
  - String name
  - ArrayList<List> lists
